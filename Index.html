SYSTEM LAUNCHER v1.0 READY

System Optimizer

ปรับแต่งการจัดการทรัพยากร CPU และความเสถียรของ Kernel

CPU Power Steering บังคับให้ Windows กระจายงานลงกลุ่ม Threads ประสิทธิภาพสูงเพื่อลด Micro-stutter
RAM Cache Purge เคลียร์ Standby List และคืนพื้นที่ Memory ที่ค้างอยู่ในระบบทั้งหมด

Deep Cleaner

ล้างไฟล์ขยะลึกระดับระบบและลบประวัติ Log ตกค้าง (Anti-Forensics)

BAM / DAM Clearer ล้างประวัติการเปิดแอปพลิเคชัน (Background Activity Moderator) ใน Registry
SRUM Log Purge ลบฐานข้อมูลการใช้งานพลังงานและเน็ตเวิร์กของระบบเพื่อความเป็นส่วนตัว

Latency Tweaks

ปรับแต่งความเร็วการตอบสนองของเมาส์ คีย์บอร์ด และเครือข่าย

Mouse Queue Optimization ปรับแต่งค่า MouseDataQueueSize เพื่อเพิ่มความสมูทและลดดีเลย์การลากเมาส์
